# 快速开始 — ChronoCoding 通用适配版

## 概述

ChronoCoding 是 AI 编程智驾框架，适配任意支持自定义规则/技能的 AI IDE（Qoder、Trae、Cursor 等）。

原 Claude Code 专属特性已改造为通用串行模式，保留全部核心能力。

## 安装

### 1. 克隆/下载本框架

```bash
git clone <repo-url> chrono-coding
cd chrono-coding
```

### 2. 运行安装脚本

**Windows (PowerShell):**
```powershell
.\install.ps1 -IdeName qoder
```

**macOS/Linux:**
```bash
./install.sh --ide qoder
```

支持的 IDE: `qoder`, `trae`, `cursor`, `vscode`, `windsurf`

### 3. 配置 API Key

复制 `model-config.json` 并填入你的 API Key：

```json
{
  "providers": {
    "deepseek": {
      "apiKey": "sk-your-deepseek-key"
    },
    "minimax": {
      "apiKey": "your-minimax-key"
    },
    "kimi": {
      "apiKey": "your-kimi-key"
    }
  }
}
```

### 4. 验证安装

```bash
node {IDE_ROOT}/helpers/hammer-bridge.cjs status
```

## 核心触发词

| 触发词 | 功能 |
|--------|------|
| `/夯` | 启动多团队竞争评审（红蓝绿三队串行执行） |
| `/triple` | 红蓝评审三方协作 |
| `/align` | 需求对齐 |
| `/review` | 代码审查 |
| `/spec` | 生成技术规格 |
| `/research` | 自动调研 |
| `/go` | 显示流程地图（在 /夯 中） |
| `status` | 刷新进度看板 |

## 目录结构

```
{IDE_ROOT}/
├── skills/           # 技能目录（21+ 个技能）
├── helpers/          # 辅助脚本（28+ 个）
├── rules/            # 全局规则
├── templates/        # 配置模板
├── memory/           # 记忆系统
└── model-config.json # 模型配置
```
